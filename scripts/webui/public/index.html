      修复 / 维护动作

      点一下就在后台启动任务，日志流到下方或「运行日志」Tab。

      定时任务 (systemd user timer)

      让脚本每天自动跑。仅 Linux 可用。关机后还想触发需要 sudo loginctl enable-linger $USER 一次。

      每日任务 (microsoft-rewards.timer)

      参考：*-*-* 07:00:00（每天 7:00）*-*-* 07,19:00:00（每天 7 和 19 点）

      管理页自启 (microsoft-rewards-webui.service)

      装上后这个管理页本身会开机自启、异常自动重启。之后连 ./manage.sh 都不用敲。
